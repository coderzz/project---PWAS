How to use the tools.

STEP I:

SEARCH BOX

- The search box displays suggestion after user types at-least 2 alphabets.
- The search box displays suggestion in in 3 levels the following order(top-down):
State (at most 10)
District, State (at most 10)
Block, District, State (at most 10)

(Example)

STEP II:

RANGE SLIDER(setting the milestone)

Set the minimum(left) and maximum(right) range of percentage of piped water supply according to which only those habitation are listed that are in the range.

STEP III:

Click on search button.
